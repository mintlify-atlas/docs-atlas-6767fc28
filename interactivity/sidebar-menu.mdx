Brasa includes a slide-in product panel that gives users a quick look at the menu from anywhere on the page. It is triggered by a hamburger-style button and closes via a close button, an overlay click, or by selecting a menu item.

## Structure overview

The panel is made up of three main pieces:

| Element | ID | Role |
|---|---|---|
| Hamburger trigger | `menuTrigger` | Opens and closes the panel |
| Dark overlay | `menuOverlay` | Covers the page; click to close |
| Slide-in panel | `menuPanel` | Contains the product list |

```html index.html
<div class="prod-menu-trigger" id="menuTrigger" onclick="toggleProdMenu()">
  <span></span>
  <span></span>
  <span></span>
</div>

<div class="prod-menu-overlay" id="menuOverlay" onclick="toggleProdMenu()"></div>

<aside class="prod-menu-panel" id="menuPanel">
  <!-- panel content -->
</aside>
```

The three `<span>` elements inside `.prod-menu-trigger` form the three lines of the hamburger icon. CSS animates them into an X when the `.open` class is applied.

## `toggleProdMenu()`

```javascript index.html
function toggleProdMenu() {
  const panel = document.getElementById('menuPanel');
  const overlay = document.getElementById('menuOverlay');
  const trigger = document.getElementById('menuTrigger');
  panel.classList.toggle('open');
  overlay.classList.toggle('open');
  trigger.classList.toggle('open');
}
```

Every call to `toggleProdMenu()` toggles the class `open` on all three elements simultaneously:

- **`menuPanel`** — sliding the panel into view (controlled by CSS `transform` or `right` property).
- **`menuOverlay`** — making the dark backdrop visible.
- **`menuTrigger`** — animating the hamburger lines into a close (X) icon.

<Note>
`toggleProdMenu()` is defined inside a `<script>` tag in `index.html`, not in `main.js`. It must remain accessible globally so that the `onclick` attributes in the HTML can reach it.
</Note>

## The overlay

The overlay is a full-screen `<div>` that sits between the page content and the panel. Its `onclick="toggleProdMenu()"` means clicking anywhere outside the panel closes it — a standard UX pattern for drawers.

The overlay is styled via CSS rules in `style.css`. It remains invisible until `.open` is applied, at which point it becomes visible and receives pointer events so clicks reach it (triggering the close).

## `scrollToGrid()`

Clicking any product item calls `scrollToGrid()` before closing the panel:

```javascript main.js
function scrollToGrid() {
  const grid = document.getElementById('menu-grid');
  if (grid) {
    grid.scrollIntoView({ behavior: 'smooth', block: 'start' });
  }
}
```

<Note>
`scrollToGrid()` targets `id="menu-grid"`. In the current `index.html`, the product grid section uses `id="grid"` (not `id="menu-grid"`). This means `scrollToGrid()` currently has no effect — the element is not found. To fix this, either rename the grid's ID in `index.html` to `menu-grid`, or update the function in `main.js` to use `getElementById('grid')`.
</Note>

The `if (grid)` check prevents a JavaScript error if the grid element is not present on the page.

## Adding a new product

To add a product to the sidebar panel, insert a new `<li>` inside `.prod-menu-list`:

```html index.html
<li onclick="scrollToGrid(); toggleProdMenu()">
  <span class="prod-menu-name">Doble Smash</span>
  <span class="prod-menu-price">$29.000</span>
</li>
```

<Tip>
The sidebar panel is a quick-access preview, not the full menu. Keep the list short. Use the "Ver menú completo" CTA at the bottom of the panel to direct users to the full product grid.
</Tip>

## Changing the panel layout

The panel's visual structure is controlled entirely by CSS classes on the `<aside>` and its children.

| Element | Class | Controls |
|---|---|---|
| Slide-in panel | `.prod-menu-panel` | Width, position, background, transform |
| Category label | `.prod-menu-label` | Uppercase tag above the title |
| Panel title | `.prod-menu-title` | Large heading with italic `<em>` |
| Product list | `.prod-menu-list` | List layout and item spacing |
| Item name | `.prod-menu-name` | Left-aligned product name |
| Item price | `.prod-menu-price` | Right-aligned price |
| CTA link | `.prod-menu-cta` | Bottom call-to-action button |

To change the panel width, update `.prod-menu-panel` in your CSS:

```css style.css
.prod-menu-panel {
  width: 340px; /* increase or decrease as needed */
}
```

## `setFilter()` and `filterProducts()`

Two additional functions in `main.js` are available for sidebar filter interactions. They are not wired up in the current `index.html` but can be used if you add a filter sidebar.

**`setFilter(el, cat)`** — marks a filter list item as active and triggers product filtering:

```javascript main.js
function setFilter(el, cat) {
  document.querySelectorAll('.sidebar-filters li').forEach(li => {
    li.classList.remove('active');
  });
  el.classList.add('active');

  const grid = document.getElementById('menu-grid');
  if (grid) {
    grid.scrollIntoView({ behavior: 'smooth', block: 'start' });
  }

  filterProducts(cat);
}
```

**`filterProducts(cat)`** — shows or hides `.product` elements based on their `data-cat` attribute:

```javascript main.js
function filterProducts(cat) {
  const products = document.querySelectorAll('.product');

  products.forEach(product => {
    const productCat = product.getAttribute('data-cat');

    if (cat === 'todas' || productCat === cat) {
      product.style.display = 'flex';
      // animate in
      product.style.opacity = '0';
      product.style.transform = 'translateY(12px)';
      setTimeout(() => {
        product.style.transition = 'opacity .4s ease, transform .4s ease';
        product.style.opacity = '1';
        product.style.transform = 'translateY(0)';
      }, 50);
    } else {
      product.style.display = 'none';
    }
  });
}
```

To use filtering, add a `data-cat` attribute to each `.product` element and call `setFilter(this, 'category')` from filter buttons with class `sidebar-filters`:

```html index.html
<ul class="sidebar-filters">
  <li onclick="setFilter(this, 'todas')">Todas</li>
  <li onclick="setFilter(this, 'clasica')">Clásicas</li>
  <li onclick="setFilter(this, 'especial')">Especiales</li>
</ul>
```

<Note>
`filterProducts('todas')` shows all products regardless of their `data-cat` value.
</Note>
