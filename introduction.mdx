## What is Brasa?

**Brasa** is a premium burger restaurant brand built around a single conviction: great food doesn't need to be complicated — it just needs the best ingredients. This website is the brand's digital presence: an editorial-style single-page experience that showcases the menu, tells the brand story, and makes it easy for customers to find and contact the restaurant.

The site is built with pure HTML, CSS, and vanilla JavaScript — no frameworks, no build tools. It's designed to be fast, beautiful, and easy to customize.

## Technology stack

| Layer | Technology |
|---|---|
| Markup | HTML5 (`index.html`) |
| Styles | CSS3 with custom properties (`style.css`, `landing.css`) |
| Interactivity | Vanilla JavaScript (`main.js`) |
| Typography | Cormorant Garamond (serif headings), DM Sans (body) via Google Fonts |
| Images | Local `.JPG` files in the `img/` directory |

## Project structure

```
burguer-brasa/
├── index.html        # Main page — all sections in one file
├── style.css         # Core design system (variables, nav, grid, components)
├── landing.css       # Landing-specific layouts (about, contact, footer)
├── main.js           # All JavaScript behavior
└── img/              # Product and hero images
    ├── hero.JPG
    ├── menu-large.JPG
    ├── about.JPG
    ├── burguer-1.JPG
    ├── burger-2.JPG
    ├── burguer-3.JPG
    └── burguer-4.JPG
```

<Note>
The project uses no build system, package manager, or framework. Everything runs directly in the browser from static files — making it trivial to host anywhere.
</Note>
