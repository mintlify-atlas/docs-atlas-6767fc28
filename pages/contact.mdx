The Contact section displays all the information a customer needs to visit or reach Brasa: physical address, opening hours, WhatsApp number, and email. It's designed for walk-in customers — no reservation system, no form.

## HTML structure

```html index.html
<section id="contacto" class="landing-contacto reveal">
  <p class="sec-label">Encuéntranos</p>
  <h2 class="contact-title">Visítanos<br><em>cuando quieras</em></h2>
  <p class="contact-note">
    Sin reserva. Sin protocolo. Solo buen ambiente y mejor comida
    — de martes a domingo.
  </p>

  <div class="info-row">
    <!-- SVG icon -->
    <div>
      <span class="info-lbl">Dirección</span>
      <span class="info-val">Tu dirección aquí — Ciudad</span>
    </div>
  </div>
  <!-- more info-row blocks -->
</section>
```

## Contact fields

| Field | Class | Current value |
|---|---|---|
| Address | `info-val` | Tu dirección aquí — Ciudad |
| Hours | `info-val` | Mar–Dom · 12:00 pm – 10:00 pm |
| WhatsApp | `info-val` | +57 300 000 0000 |
| Email | `info-val` | hola@brasa.co |

## Updating contact details

All four contact fields are in `index.html` inside `.info-row` blocks. Edit the `.info-val` span for each:

```html index.html
<!-- Address -->
<span class="info-val">Calle 123 #45-67 — Bogotá, Colombia</span>

<!-- Hours -->
<span class="info-val">Mar–Dom · 12:00 pm – 10:00 pm</span>

<!-- WhatsApp -->
<span class="info-val">+57 310 123 4567</span>

<!-- Email -->
<span class="info-val">contacto@brasa.co</span>
```

<Note>
There is no contact form — customers are expected to reach out via WhatsApp or email directly. This keeps the page simple and works well for small restaurants.
</Note>

## Making WhatsApp clickable

The WhatsApp number is currently plain text. To make it a clickable link that opens WhatsApp directly, wrap it in an anchor tag:

```html index.html
<div>
  <span class="info-lbl">WhatsApp</span>
  <a href="https://wa.me/573001234567" class="info-val">+57 300 123 4567</a>
</div>
```

Replace `573001234567` with your country code + number (no spaces, no +).

## Making email clickable

Similarly, turn the email into a `mailto:` link:

```html index.html
<div>
  <span class="info-lbl">Correo</span>
  <a href="mailto:hola@brasa.co" class="info-val">hola@brasa.co</a>
</div>
```

## Adding a map embed

To add a Google Maps embed below the contact details, paste the embed iframe from Google Maps inside the section:

```html index.html
<div class="map-embed" style="margin-top: 3rem; border-radius: 8px; overflow: hidden;">
  <iframe
    src="https://www.google.com/maps/embed?pb=YOUR_EMBED_CODE"
    width="100%"
    height="300"
    style="border:0;"
    allowfullscreen=""
    loading="lazy">
  </iframe>
</div>
```

<Tip>
Get your embed code from Google Maps: search your address → Share → Embed a map → Copy HTML.
</Tip>
