## Prerequisites

You only need a web browser and a way to serve static files. No Node.js, no build step, no dependencies.

## Run locally

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/LeynerDurk/burguer-brasa.git
    cd burguer-brasa
    ```
  </Step>
  <Step title="Serve the files">
    Open `index.html` directly in your browser, or use a local server for a better development experience:

    <CodeGroup>
    ```bash Python
    python3 -m http.server 8080
    ```
    ```bash Node.js (npx)
    npx serve .
    ```
    ```bash VS Code
    # Install the "Live Server" extension, then right-click index.html → Open with Live Server
    ```
    </CodeGroup>
  </Step>
  <Step title="View the site">
    Navigate to `http://localhost:8080` in your browser. You should see the full Brasa editorial landing page with:

    - The full-screen hero image
    - The product grid with all four burgers
    - The animated scroll reveal effects
    - The slide-in product menu panel
  </Step>
</Steps>

## What you'll see

Once the site loads, here's what each part of the page does:

| Section | Description |
|---|---|
| Hero | Full-screen `hero.JPG` with the tagline "El sabor que queda." |
| Brasa Signature feature | Split layout with `menu-large.JPG` and product description |
| "A la brasa" wide banner | Full-width `about.JPG` with overlay text |
| Product grid | Four burger cards with Add to Order buttons |
| CTA | "El mejor sabor te espera" call to action |
| Nosotros | Brand story, four pillars, and stats |
| Contacto | Address, hours, WhatsApp, and email |

## Make your first customization

Update the restaurant's contact details in `index.html`. Find the Contacto section and replace the placeholder values:

```html index.html
<!-- Address -->
<span class="info-val">Tu dirección aquí — Ciudad</span>

<!-- Hours -->
<span class="info-val">Mar–Dom · 12:00 pm – 10:00 pm</span>

<!-- WhatsApp -->
<span class="info-val">+57 300 000 0000</span>

<!-- Email -->
<span class="info-val">hola@brasa.co</span>
```

<Tip>
After editing `index.html`, refresh your browser to see the changes immediately — there's no build step required.
</Tip>
